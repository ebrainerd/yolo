# Neon Nocturne

Immersive first-person night city you can run around in. Talk to NPCs. They will say philosophical things for no reason.

## Run

```bash
npm install
npm run dev
```

Open the local URL. On the Neon Nocturne start screen, click **Click to enter** to lock the pointer, then:

- **WASD** — move
- **Shift** — sprint
- **Mouse** — look
- **E** — talk / continue dialogue when prompted
- **Esc** — release pointer
